# Obscura

*Less webpack, more f‑stop!*
*You can visit the project homepage here: https://obscura.md*

![Screenshot of an example Obscura site](/docs/assets/demo-site-01.jpg)

A purpose-built static site generator for photography portfolio websites. It takes your photos, organises them into galleries, generates responsive images, and produces a fast static site with a lightbox viewer, blog with photo shortcodes, and cross-referenced content — all from Markdown files and YAML config.
