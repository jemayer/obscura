At build time, scan all gallery directories for photos without a corresponding .yaml sidecar. Generate a sidecar pre-filled with EXIF data and empty fields for manual enrichment. Never overwrite existing sidecars. Include the '# Auto-generated by comagen' header comment. Per ADR-010.

## Summary of Changes

- Created src/sidecar.ts with:
  - generateSidecars(): scans a gallery directory, generates YAML sidecars for photos that don't have one
  - generateAllSidecars(): runs across all gallery directories
  - Pre-fills sidecars with EXIF data (date, camera, lens, GPS), leaves empty fields for manual enrichment
  - Never overwrites existing sidecars (per ADR-010)
  - Includes '# Auto-generated by comagen' header comment
  - isPhotoFile(): validates supported extensions
  - sidecarPathForPhoto(): derives sidecar path from photo path
  - Collects and returns EXIF warnings for later display
