This is a photography portfolio built with [Obscura](https://github.com/jemayer/obscura), a static site generator made for photographers only.

## Getting Started

Replace this page with your own content by editing `site/content/pages/about.md`. You can write anything here using Markdown — your bio, contact information, exhibition history, or equipment list.

## How This Site Works

- **Photos** live in `site/content/photos/`, organised into galleries
- **Blog posts** are Markdown files in `site/content/posts/`
- **Pages** like this one are in `site/content/pages/`
- Run `npm run build` to generate the site
- Run `npm run dev` to preview with live reload
