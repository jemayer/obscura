<u>**Obscura** is a purpose-built static site generator for photography portfolio websites.</u> It takes a directory of pictures organised into galleries, parses blog posts and simple pages formatted in Markdown, and produces a fully static website with responsive images, a lightbox viewer, extracted EXIF data, and cross-referenced content.

**Obscura** isn't trying to be everything. It's made for photographers putting their work on the web. Every decision made and every feature included serves that one single purpose. The CLI tool to support your image metadata workflow? Same deal - no fluff, just function. <u>Bottom line: Less webpack, more f-stop!</u>
